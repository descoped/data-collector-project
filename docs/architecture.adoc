== Pre-requisites

* Java 11 or 12. Because of GraalVM dependencies, Java 13 is yet not supported.
* Docker CE 18+
* Docker Compose 1.2x+

== Modules

The Data Collector uses JPMS (module system) for strong encapsulation, security and maintenance.
